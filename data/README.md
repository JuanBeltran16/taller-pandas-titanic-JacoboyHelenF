## Objetivo

Explorar y analizar el dataset del Titanic mediante herramientas de 
ciencia de datos como Python y Pandas. El proyecto busca responder 
preguntas concretas sobre los pasajeros del Titanic, sus caracteristicas 
demograficas y sus probabilidades de supervivencia, aplicando tecnicas 
de analisis estadistico y visualizacion de datos.

## Estructura del proyecto

taller-pandas-titanic-JacoboyHelenF/
├── data/           # Instrucciones de descarga de los datos
├── notebooks/      # Notebook principal del análisis
├── src/            # Scripts de Python
├── outputs/
│   ├── tablas/     # Resultados en CSV
│   └── graficos/   # Visualizaciones
└── docs/           # Informe final

## Cómo ejecutar

1. Clonar el repositorio
2. Descargar los datos siguiendo las instrucciones en data/README.md
3. Instalar dependencias: pip install pandas matplotlib seaborn
4. Abrir notebooks/proyecto_titanic.ipynb con Jupyter
