# Datos del proyecto

Los archivos de datos originales NO están incluidos en este repositorio.

## Cómo descargarlos

1. Ir a https://www.kaggle.com/competitions/titanic
2. Crear una cuenta gratuita en Kaggle
3. Ir a la pestaña "Data" y descargar:
   - train.csv
   - test.csv
4. Colocar los archivos descargados en esta carpeta (data/)
